# InfiniDrive GUI



   This project provides a GUI for the InfiniDrive functionality using Flet.



   ## Installation



   ```sh

   git clone https://github.com/nicomda/InfiniDrive

   cd InfiniDrive

   pip install -r requirements.txt

   ```



   ## Usage



   ```sh

   python main.py

   ```



   ```



### 実行手順

1. リポジトリをクローンし、ディレクトリに移動します。

   ```sh

   git clone https://github.com/nicomda/InfiniDrive

   cd InfiniDrive

   ```



2. 依存関係をインストールします。

   ```sh

   pip install -r requirements.txt

   ```



3. アプリケーションを実行します。
